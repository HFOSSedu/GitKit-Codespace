The command
git pull --ff-only <remote> <branch>
will pull any new commits from the specified branch in the remote repo and add them onto the end of the active branch of your local repo. <branch> can be a name like origin. You will need to modify the git pull command above to pull and add the commits from the main branch of the upstream repo to your main branch.
