The git mergetool command will launch the VSCode merge tool so that you can use it to resolve the conflict.
To open a file in VS Code’s Merge Editor:
(a)
Issue the git mergetool command.
When VS Code’s Merge Editor is first opened, it will open a window containing three panes (LOCAL, REMOTE and Result). The content that you see will be different, but the window that you see should be similar to the one shown in Figure 5.4.2.
(b)
Open the conflicted file, and click the "Resolve in Merge Editor" ( Figure 5.4.3) button in the lower right corner of the editor pane.
When VS Code’s Merge Editor is first opened, it will open a window containing three panes (LOCAL, REMOTE and Result). The content that you see will be different, but the window that you see should be similar to the one shown in Figure 5.4.4.
